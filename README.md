# Identity Builder App

**Identity Builder** is a React Native + Expo mobile app that helps users define their goals and generates identity suggestions using AI. This project serves as a case study demonstrating fullstack AI integration.

---

## Demo

![Identity Builder Demo](assets/demo.gif)

---

## Features

- Input a **specific goal**.  
- Generate **3 AI-powered identity suggestions**.  
- Clean and simple **React Native UI**.  
- Backend securely handles AI requests; API key is never exposed to the frontend.  

---

## Tech Stack

- **Frontend:** React Native + Expo  
- **Backend:** Node.js + Express  
- **AI:** Groq / OpenAI API  
- **Deployment:** Local development or cloud (Vercel recommended)  

## Getting Started

- Clone the repo
- Install dependencies for frontend and backend
- Create a .env file with your AI key
- Start backend server
- Run Expo frontend
